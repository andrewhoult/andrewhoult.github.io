Shadows Beneath the Dust is a multiplayer co-op horror game inspired by Lethal Company.

Steam Page

Notes:

  • Development time: 6 months
  • Group size: 6
  • Date: October 2024 - March 2025

What I Did:

  • Created a custom netcode library in C# based on Steam Net Sockets and UDP/TCP
  • Integrated Steam lobbies, freinds lists, achievements
  • Created a procedural generation algorithm for mines, ensuring a traversable path always exists to the exit
  • Created a custom deferred rendering pipeline in Unity that uses mesh-based shadow volumes
  • Created a fully networked inventory system
  • Created a networked first/thirdperson animation system supporting multiple layers and masks for holding various items, looking around, etc
  • Created a system for proximity chat with environment-based reverb and occlusion
  • Created textures, sounds, and music

Tools Used

Created in Unity, uses Steamworks API.


Development

Challenges:

  • Creating a system to easily sync all game data, especially on connection midway through a match.
  • Fast rendering of large procedurally generated environments and lots of real-time shadow-casting lights.
  • Generating interesting procedural mines.
  • Minimizing bandwidth usage.


Features (as of now):

  • Custom netcode built using Steam Networking Sockets and TCP (for LAN mode)
  • Proximity chat
  • Easy game joining and integration with Steam friends
  • Achievements and other stats
  • Procedurally generated mines
  • Custom render pipeline with stencil based realtime shadows
  • Deferred renderer optimized for our game
  • Fully synced inventory system

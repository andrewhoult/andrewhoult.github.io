This is a simple 3D engine programmed from scratch in Typescript. I started this project as an excuse to learn how to program for the web, and eventually abandoned it as the scope grew too big.

Source Code


Notes:

  • Solo
  • Date: 2023
  • Duration: On and off for a few months

Features:

  • Continuous player collision
  • Multiplayer
  • Client-server architecture (server authoritative)
  • Client-side prediction
  • Basic map editor
  • GLTF model loader
  • Skeletal vertex deformation and animations

*Assets are from Half Life and Duke Nukem
